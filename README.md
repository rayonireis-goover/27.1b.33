## Contribuindo

Este repositório é muito pequeno e contém apenas arquivos estáticos. Antes de abrir um PR, considere:

- Fazer mudanças pequenas e focadas (correções de formatação, ortografia, metadados ou acessibilidade de HTML).
- Atualizar o `README.md` quando fizer alterações significativas para documentar o propósito.
- Evitar adicionar dependências, ferramentas de build, ou infra sem orientação do mantenedor.

As instruções para agentes AI estão em `.github/copilot-instructions.md` — leia antes de fazer mudanças automáticas.

Se estiver em dúvida sobre a direção do projeto (ex.: transformar em site/demos vs manter estático), abra uma issue ou consulte o mantenedor primeiro.
